# ガチャシミュレーター

このゲームは、ガチャを引くだけのゲームです。

## 概要

このゲームは、ソシャゲのガチャのようないろんな演出が用意されています。ぜひ10連で最高レアを当ててください！

犬の画像が出てくる**ワンダフル冒険ガチャ**と猫の画像が出てくる**ニャンダフル究極の猫ガチャ**の二つのガチャから
選ぶことができます。

- **ジャンル**: カジュアル
- **ターゲット**: 全てのプレイヤー
